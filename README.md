# Evaluna Nails Spa 2.0

Sitio web oficial con diseño futurista elegante para manicura boutique en Chía.

## Características

- **Modo oscuro/claro** con toggle y persistencia en localStorage
- **Cursor personalizado** con efecto de estela
- **Partículas animadas** en el hero
- **Glassmorphism** en tarjetas y superficies
- **Animaciones WOW**: scroll reveals, text reveal, 3D tilt, parallax, botones magnéticos
- **Lightbox** para galería de imágenes
- **Navbar dinámico** que se oculta al hacer scroll
- **Loader** de entrada con animación
- **Diseño responsive** para todos los dispositivos
- **Optimizado para accesibilidad** y prefers-reduced-motion
